How to install : 

1 : Edit theme.html. In line 65 edit those links and replace theme by yours.

<!-- Header bar Links : You must replace those links by yours.-->
<div id="menubar"><a href="<{$xoops_url}>/modules/articles/">» Articles</a> <a href="<{$xoops_url}>/modules/newbb/">» Forums</a> <a href="<{$xoops_url}>/modules/wordpress/">» Wordpress</a> <a href="<{$xoops_url}>/modules/downloads/">» Download</a> <a href="<{$xoops_url}>/modules/planet">» Planet</a> <a href="<{$xoops_url}>/modules/contact/">» Contact</a></div>
<!-- End of Header bar Links-->

2 : With a picture editor, edit the file Logo.psd and save it as logo.gif

3 : Upload the folder (ex : /nature/) to your themes folder (www.xoopssite.com/themes/)

4 : Go to your admin interface and select your new theme.

5 : Enjoy.
